___________________________ FONCTIONS UTILISATEURS

Login :
	• permet de se connecter à sa session de clavardage en s'authentifiant
	• bouton "se connecter" de interface graphique
	• néant

Se_Déconnecter : 
	• permet de se déconnecté et de se marqué indisponible
	• bouton "se déconnecter" de interface graphique
	• utilisateur authentifié 

Choisir_pseudo : 
	• permet de modifier son pseudo en un autre 
	• bouton "modifier pseudo" de interface graphique
	• utilisateur authentifié  

Discuter : 
	• permet à un utilisateur d'engager une conversation avec un autre
	• bouton "choisir destinataire" de interface graphique
	• utilisateur émetteur et destinataires connectés 


___________________________ FONCTIONS ADMINISTRATEURS

Ajouter_compte : 
	• permet d'ajouter un compte utilisateur caractérisé par un pseudo unique
	• bouton "ajouter compte" de interface graphique
	• utilisateur authentifié et reconnu comme admin 
 
Supprimer_compte : 
	• permet de supprimer un compte utilisateur non admin caractérisé par un pseudo unique
	• bouton "supprimer compte" de interface graphique
	• utilisateur authentifié et reconnu comme admin  

Créer_groupe : 
	• permet de créer un nouveau groupe par son nom 
	• bouton "créer groupe" de interface graphique
	• utilisateur authentifié et reconnu comme admin 

Supprimer_groupe : 
	• permet de supprimer un groupe choisi
	• bouton "supprimer groupe" de interface graphique
	• utilisateur authentifié et reconnu comme admin 

Ajouter_utilisateur_groupe : 
	• permet d'ajouter un compte utilisateur, caractérisé par un pseudo unique, à un groupe existant
	• bouton "ajouter au groupe" de interface graphique
	• utilisateur authentifié et reconnu comme admin 

Supprimer_utilisateur_groupe : 
	• permet dde supprimer un compte utilisateur, caractérisé par un pseudo unique, à un groupe existant
	• bouton "supprimer du groupe" de interface graphique
	• utilisateur authentifié et reconnu comme admin 

Changer_mot_de_passe : 
	• permet de changer le mot de passe d'un utilisateur
	• bouton "changer mot de passe" de interface graphique
	• utilisateur authentifié et reconnu comme admin
